# Proyecto: Sitio Web "El Antojo"

## Descripción del Tema

Este proyecto es un sitio web para un restaurante ficticio llamado "El Antojo". El concepto se basa en un restaurante de barrio tradicional y rústico, un "bodegón" ubicado en Madrid, que lleva sirviendo comida típica española desde 1950.

El diseño y el contenido buscan evocar una sensación de calidez, tradición y autenticidad, como si fuera un negocio familiar con una larga historia en la ciudad.

## Tecnologías Utilizadas

Para la realización de este proyecto (Parte 1 del enunciado), se han utilizado las siguientes tecnologías:

*   **HTML5:** Para la estructura y el contenido semántico de las páginas web.
*   **CSS3:** Para el diseño, el estilo visual y la maquetación. Se ha implementado un diseño responsivo utilizando Flexbox, CSS Grid y Media Queries.

No se ha utilizado JavaScript, siguiendo el enfoque opcional del enunciado para esta primera parte.

## Estructura del Proyecto

El proyecto se organiza de la siguiente manera:

```
/
├── index.html         // Página de Inicio
├── menu.html          // Página con el menú del restaurante
├── dashboard.html     // Página del panel de datos
├── css/
│   └── style.css      // Hoja de estilos principal (incluye estilos responsivos)
└── README.md          // Este archivo
```

*   **`index.html`**: La página principal que presenta el restaurante.
*   **`menu.html`**: Muestra el menú completo, dividido en categorías.
*   **`dashboard.html`**: Contiene el panel de datos del restaurante.
*   **`css/style.css`**: Archivo único de estilos que define la apariencia de todo el sitio, incluyendo las reglas para adaptarse a diferentes tamaños de pantalla (diseño responsivo).
